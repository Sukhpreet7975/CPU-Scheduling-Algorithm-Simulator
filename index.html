CPU Scheduling Algorithm Simulator

Visualize and Compare Common CPU Scheduling Techniques

Objective

This web application provides a hands-on tool for visualizing and understanding various non-preemptive and preemptive CPU scheduling algorithms. Users can input process details (arrival time, CPU/IO burst sequence, priority) and observe how different algorithms manage these processes in real-time, including context switch overhead. The simulator calculates key performance metrics (Waiting Time, Turnaround Time, Response Time, Throughput), displays results, generates a dynamic Gantt chart, shows simulated system resource usage, and provides a step-by-step trace of scheduler decisions.

The primary goal is to serve as an educational resource for students learning operating systems concepts, allowing them to experiment with different inputs and compare the behaviour and efficiency of algorithms like FCFS, SJF, SRTF, Priority (NP/P), and Round Robin, incorporating factors like I/O waits and context switches.

Technologies Used

  • HTML5: Structures the content of the web pages.
  • CSS3: Styles the application for a clean interface, layout (Flexbox), real-time visualization, and Gantt chart rendering.
  • JavaScript (Vanilla): Implements all client-side logic:
    • Dynamic input field generation.
    • Input validation (including burst sequence parsing).
    • Real-time simulation engine (`setInterval`).
    • Scheduling algorithm logic (FCFS, SJF, SRTF, Priority NP/P, RR).
    • Handling of I/O bursts and blocked state.
    • Simulation of context switch overhead.
    • Performance metric calculations (Wait, TAT, Response, Throughput).
    • DOM manipulation for results, Gantt chart, system monitor, and trace updates.
    • Event handling for controls, inputs, save/load, presets.
    • Basic state saving/loading with `localStorage`.

Current Features

  • Input process details (Arrival Time, CPU/IO Burst Sequence, Priority).
  • Input for Context Switch Time.
  • Select from various scheduling algorithms:
    • First-Come, First-Served (FCFS)
    • Shortest Job First (SJF) (Non-Preemptive - based on next CPU burst)
    • Shortest Remaining Time First (SRTF) (Preemptive SJF - based on remaining CPU burst)
    • Priority (Non-Preemptive)
    • Priority (Preemptive)
    • Round Robin (RR)
  • Real-time simulation visualization with Play/Pause/Reset/Speed controls.
  • Dynamic Gantt chart generation including 'Idle' and 'CS' (Context Switch) periods.
  • Live updates of the results table (Status, Wait, TAT, Response, Completion) as processes progress/finish.
  • Calculation and display of average metrics upon completion.
  • Simulated System Monitor showing CPU and Memory utilization.
  • Step-by-step execution trace panel.
  • Save and Load process configurations to/from browser storage.
  • Preset process examples.
  • Light/Dark theme toggle.

  • **Advanced Gantt Chart:** Zooming, panning, better visual state representation.
  • **Detailed Queue Visualization:** Animated movement of processes between states/queues.
  • **Aging Implementation:** Add aging to priority or MLFQ algorithms.
